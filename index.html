Array State Operations

ADD object 
setItems(prev => [
  ...prev,
  newItem
]);
DELETE by id 
setItems(prev =>
  prev.filter(item =>
    item.id !== id
  )
);
TOGGLE status 
setItems(prev =>
  prev.map(item =>
    item.id === id
      ? { ...item,
          active: !item.active }
      : item
  )
);
💡 Rule of thumb: Always use prev => callback form. Never mutate state directly. Spread ...item on toggle — never mutate the object itself.

useEffect Patterns

Async fetch inside useEffect (correct pattern) 
useEffect(() => {
  let cancelled = false;

  const fetchData = async () => {
    const response = await fetch("https://api.example.com/user");
    const data = await response.json();  // double await!
    if (!cancelled) setUser(data);       // guard unmounted
  };

  fetchData();  // don't forget to invoke!

  return () => { cancelled = true; };  // cleanup here, not inside async fn
}, []);
❌ Wrong — async directly on useEffect 
// React expects: undefined | () => void
// async fn returns a Promise — React ignores it, cleanup is lost
useEffect(async () => {        // ← NEVER do this
  const data = await fetchSomething();
  setData(data);
}, []);
